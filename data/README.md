# Data Directory

Этот каталог хранится в репозитории как служебная точка входа для веб-статики проекта.

## Размещение статики

- Исходные или собранные файлы фронтенда кладутся в `data/dist`.
- Эти файлы раздаются устройством через `SPIFFS`.
- Содержимое `data/dist` не должно коммититься в репозиторий.

## Назначение

Каталог `data` оставлен в репозитории как tracked-папка для понимания, где должен лежать фронтенд для веб-сервера и как он подхватывается прошивкой.
