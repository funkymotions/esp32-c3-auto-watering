# ESP32-C3 Auto Watering

Проект автоматического полива на ESP32-C3.

Устройство поднимает собственную точку доступа для первичной настройки, сохраняет параметры в NVS, при необходимости синхронизирует время по Wi-Fi и затем работает по циклу: включает насос на заданное время, уходит в глубокий сон и просыпается по расписанию.

## Цель проекта

Автоматически поливать растение по заданному расписанию без постоянной работы контроллера и без ручного запуска.

## Архитектура

- `src/main.cpp` — основной жизненный цикл устройства: инициализация, окно конфигурации, синхронизация времени, запуск насоса и переход в deep sleep.
- `src/webserver.cpp` — HTTP API для чтения и изменения параметров.
- `SPIFFS` — раздача веб-интерфейса из `data/dist`.
- `Preferences` — хранение настроек во flash памяти.
- `ESPAsyncWebServer` — асинхронный веб-сервер для локальной настройки.
- `NTP` через внешний Wi-Fi — синхронизация системного времени перед расчётом следующего пробуждения.

## Ключевые параметры

- `init_window` — окно первичной настройки по Wi-Fi в минутах; значение эффективно подхватывается на лету во время этого окна.
- `pump_duration` — время работы насоса в секундах.
- `sleep_duration` — интервал между включениями в днях.
- `pump_time` — время суток следующего включения в формате `чч:мм`.
- `ext_ssid` — SSID внешней Wi-Fi сети для синхронизации времени.
- `ext_pwd` — пароль внешней Wi-Fi сети для синхронизации времени.

## Сборка

Проект собран под `PlatformIO` для платы `esp32-c3-devkitm-1`.
